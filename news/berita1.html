Kebijakan Jual Beli Uang Server - KG SURVIVAL S1

Tanggal Pembaruan: 3 April 2025

Kami di KG SURVIVAL S1 berkomitmen untuk menciptakan pengalaman bermain yang adil, aman, dan menyenangkan bagi seluruh pemain. Sebagai bagian dari upaya ini, kami ingin mengklarifikasi aturan terkait jual beli uang server.

1. Kepemilikan Uang Server

Uang server yang digunakan dalam permainan adalah properti eksklusif dari server dan dikelola sepenuhnya oleh pihak pengelola. Pemain tidak diperbolehkan untuk membeli, menjual, atau menukarkan uang server dengan mata uang nyata atau barang lain di luar platform yang telah disediakan oleh server.

2. Larangan Jual Beli Uang Server

Setiap bentuk transaksi yang melibatkan penjualan atau pembelian uang server di luar platform yang sah adalah ilegal dan melanggar ketentuan server. Hal ini mencakup, tetapi tidak terbatas pada:

  • Menjual atau membeli uang server menggunakan mata uang nyata, barang fisik, atau bentuk transaksi lainnya.
  • Penggunaan situs web atau platform pihak ketiga untuk melakukan transaksi terkait uang server.

3. Sanksi atas Pelanggaran

Pelanggaran terhadap kebijakan jual beli uang server akan dikenakan sanksi sebagai berikut:

  • Peringatan pertama: Pemain akan diberikan peringatan resmi mengenai pelanggaran yang dilakukan.
  • Ban Sementara: Jika pelanggaran berlanjut atau terjadi untuk kedua kalinya, pemain akan dikenakan ban sementara dari server, dengan durasi yang ditentukan oleh pengelola server.
  • Ban Permanen: Jika pelanggaran terus berulang atau jika ditemukan pelanggaran yang bersifat serius, pemain akan dikenakan ban permanen dari server tanpa kemungkinan pemulihan akun.
  • Penghapusan Aset: Selain hukuman di atas, pemain yang terbukti terlibat dalam jual beli uang server ilegal akan kehilangan seluruh aset berupa uang server yang ada di akun mereka. Aset yang hilang tidak akan dikembalikan.

4. Keamanan dan Keadilan

Kami ingin memastikan bahwa semua pemain dapat menikmati permainan secara adil. Kebijakan ini dirancang untuk menjaga integritas server dan mencegah ketidakseimbangan yang dapat merusak pengalaman bermain bagi seluruh pemain.

5. Pembaruan Kebijakan

Kebijakan ini dapat diperbarui dari waktu ke waktu sesuai kebutuhan untuk memastikan kelancaran dan keamanan server. Kami akan memberitahukan pemain tentang pembaruan tersebut melalui saluran komunikasi resmi server.

Dengan mematuhi kebijakan ini, Anda membantu menciptakan komunitas yang lebih baik dan lebih adil di KG SURVIVAL S1. Terima kasih atas kerjasama dan pengertiannya.
